# Браузерная игра

Небольшая браузерная игра в разработке. Концепция и игровые механики пока не определены.

## Локальный запуск

Проект не требует установки зависимостей или сборки. Откройте `index.html` в современном браузере.

При необходимости запуска через локальный HTTP-сервер можно выполнить из корня репозитория:

```sh
python3 -m http.server 8000
```

После этого откройте `http://localhost:8000`.

## Публикация

Проект рассчитан на публикацию как обычный статический сайт через GitHub Pages.

## Документация

Игровая и техническая документация находится в каталоге `docs/`:

- `GAME_DESIGN.md` — концепция и правила игры;
- `ARCHITECTURE.md` — фактическая структура и рекомендации по разделению кода;
- `TODO.md` — план дальнейшей работы;
- `CHANGELOG.md` — история изменений.

## Структура каталогов

- `css/` — стили;
- `js/` — JavaScript-код приложения;
- `js/game/` — отдельные игровые механики;
- `assets/images/` — изображения;
- `assets/audio/` — аудиоматериалы;
- `docs/` — документация;
- `tests/` — проверки проекта.
