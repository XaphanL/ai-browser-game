# Браузерная игра

Прототип браузерного roguelike top-down экшена о прохождении комнат лабиринта. Игрок в основном зачищает арены, иногда выполняет особые задания, сражается энергетическим мечом и отражает снаряды истощаемым щитом.

## Локальный запуск

Проект не требует установки зависимостей или сборки. Из-за использования JavaScript-модулей запускайте его через локальный HTTP-сервер из корня репозитория:

```sh
python3 -m http.server 8000
```

После этого откройте `http://localhost:8000`.

## Публикация

Проект рассчитан на публикацию как обычный статический сайт через GitHub Pages.

## Документация

Игровая и техническая документация находится в каталоге `docs/`:

- `GAME_DESIGN.md` — концепция и правила игры;
- `ARCHITECTURE.md` — фактическая структура и разделение кода;
- `TODO.md` — план дальнейшей работы;
- `CHANGELOG.md` — история изменений.

## Структура каталогов

- `css/` — стили;
- `js/` — JavaScript-код приложения;
- `js/` — состояние, игровая логика, ввод и отображение;
- `assets/images/` — изображения;
- `assets/audio/` — аудиоматериалы;
- `docs/` — документация;
- `tests/` — проверки проекта.
