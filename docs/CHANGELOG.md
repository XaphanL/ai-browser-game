# История изменений

## 2026-07-22

- Добавлены очки за уничтожение врагов и магазин между аренами: три случайные карточки из десяти улучшений, платный рерол, возможность продолжить без покупки и сохранение бонусов до конца забега.
- Добавлен энергетический меч: ближний секторный удар наносит 2 урона, а атака и активный щит взаимно блокируются; добавлено управление с клавиатуры, мыши и мобильной кнопки.
- Отражение снарядов сделано заметнее с помощью контактной вспышки, расширяющегося кольца и яркого следа отражённого снаряда.
- Исправлена физика направленного щита: траектория снаряда теперь отражается относительно нормали поверхности, а не задаётся направлением взгляда игрока.
- Добавлена восьмигранная броня корпуса: каждая грань выдерживает одно попадание и восстанавливается после захвата точки.
- Последовательность арен оформлена как короткий маршрут по комнатам лабиринта; после четырёх арен добавлена комната с боссом-заглушкой — крупной турелью с отдельной полосой здоровья.
- Добавлены состояние победы, новый забег после победы и отображение оставшихся граней брони в HUD.
- Создан первый играбельный прототип top-down арены: движение, захват точки с убывающим прогрессом, открытие выходов и переход между аренами разной сложности.
- Добавлены турели, снаряды, здоровье и направленный энергетический щит, отражающий снаряды и расходующий заряд.
- Добавлены адаптивный Canvas-интерфейс, HUD, управление с клавиатуры/мыши и экранные кнопки для мобильных устройств.
- Игровой код разделён на состояние, симуляцию, ввод, Canvas-отрисовку и DOM-интерфейс; обновлены игровая, архитектурная и концептуальная документация.
- README обновлён с учётом новой концепции и запуска ES-модулей через локальный сервер.
- Добавлен журнал долгосрочных решений `docs/MEMORY.md` и правила его использования.
- Уточнены правила чтения документации перед задачами: `AGENTS.md` читается всегда, а игровая и архитектурная документация — только для соответствующих задач.
- Отменены концепция «Передвижного завода» и основанный на ней первый прототип, чтобы начать проектирование игры заново.
- Подготовлена минимальная структура каталогов для дальнейшей разработки браузерной игры.
- Добавлены правила работы с репозиторием и начальная игровая, архитектурная и проектная документация.
- Обновлено описание проекта и инструкции по локальному запуску.
