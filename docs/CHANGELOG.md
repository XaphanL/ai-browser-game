# История изменений

## 2026-07-22

- Реализован первый играбельный прототип на одном адаптивном экране: ресурсы, маршрут, добыча, производство, переезды, три базовых улучшения, победа и поражение.
- Игровые данные, состояние и отображение разделены на самостоятельные JavaScript-модули без внешних зависимостей.
- Подготовлена минимальная структура каталогов для дальнейшей разработки браузерной игры.
- Добавлены правила работы с репозиторием и начальная игровая, архитектурная и проектная документация.
- Обновлено описание проекта и инструкции по локальному запуску.
- Описана концепция передвижного завода и минимальный игровой цикл без реализации механик в коде.
