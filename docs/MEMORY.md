# MEMORY

Журнал долгосрочных решений, ограничений и договорённостей проекта.

## Постоянные принципы проекта

Здесь фиксируются устойчивые принципы, определяющие направление и стиль разработки.

## Принятые решения

- 2026-07-22: игра развивается как top-down экшен о захвате точек на последовательности арен. Это новое направление прямо выбрано пользователем после отказа от предыдущей концепции.
- 2026-07-22: основная атака игрока — отражение снарядов истощаемым энергетическим щитом; обычного оружия у игрока нет. Это формирует центральную боевую механику.
- 2026-07-22: после захвата открываются выходы с заранее показанной сложностью следующей арены, чтобы игрок выбирал дальнейший риск.

## От чего отказались

## Важные договорённости
