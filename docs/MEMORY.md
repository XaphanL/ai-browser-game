# MEMORY

Журнал долгосрочных решений, ограничений и договорённостей проекта.

## Постоянные принципы проекта

Здесь фиксируются устойчивые принципы, определяющие направление и стиль разработки.

## Принятые решения

- 2026-07-22: игра развивается как top-down экшен о захвате точек на последовательности арен. Это новое направление прямо выбрано пользователем после отказа от предыдущей концепции.
- 2026-07-22: игрок сочетает энергетический меч ближнего боя с истощаемым отражающим щитом. Меч наносит 2 урона, а атака и активный щит взаимно исключаются; решение расширяет бой самостоятельной ближней атакой.
- 2026-07-22: после захвата открываются выходы с заранее показанной сложностью следующей арены, чтобы игрок выбирал дальнейший риск.
- 2026-07-22: арены становятся связанными комнатами лабиринта, по которому игрок движется к боссу. Направление выбрано пользователем, чтобы у последовательности арен появилась конечная цель и пространственный контекст.
- 2026-07-22: вокруг корпуса игрока есть восемь независимых одноразовых граней брони, восстанавливаемых захватом точки; направленный энергетический щит остаётся активным средством отражения.
- 2026-07-22: временный босс реализуется как большая турель другого цвета с полосой здоровья до появления полноценного дизайна босса.
- 2026-07-22: каждый забег использует процедурный лабиринт комнат с мини-картой пройденного пути; состояния посещённых комнат сохраняются, а магазин появляется только перед первым входом в новую арену. Решение прямо выбрано пользователем, чтобы исследование и возвраты стали частью забега.
